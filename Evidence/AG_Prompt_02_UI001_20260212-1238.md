## Design Decisions

| Decision                                      | Rationale                                                                                |
| --------------------------------------------- | ---------------------------------------------------------------------------------------- |
| **Root = Vite app** (Option A)                | Single-app repo; simplifies agent onboarding. Can restructure to monorepo later via ADR. |
| **Tailwind v4 @theme** instead of config file | v4 best practice; tokens declared in CSS, auto-generate utilities                        |
| **Motion instead of framer-motion**           | `motion` is the official v12 standalone package                                          |
| **No shadcn/ui**                              | Keeps dependency surface small; custom glass components are more on-brand                |
| **Custom SVG icons**                          | No icon library dependency; only 6 icons needed                                          |
| **CSS variables + Tailwind**                  | Variables enable runtime access; Tailwind provides utility classes                       |
